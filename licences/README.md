# Licences

This directory centralizes the licence texts for `qadgnuplotlib` and bundled third-party code.

## Project code

| Component | Licence file |
|---|---|
| qadgnuplotlib wrapper/library code | `QADGNUPLOTLIB-MIT.txt` |

## Bundled third-party code

| Component | Licence file |
|---|---|
| gnuplot | `GNUPLOT-Copyright.txt` |
| LunaSVG | `LUNASVG-MIT.txt` |
| PlutoVG | `PLUTOVG-MIT.txt` |
| PlutoVG FreeType-derived code | `FREETYPE-FTL.txt` |
| miniz | `MINIZ-MIT.txt` |
| Bob Trower base64 | `BASE64-MIT-Bob-Trower.txt` |
| stb headers | `STB_IMAGE-H-DUAL-LICENSE.txt`, `STB_IMAGE_WRITE-H-DUAL-LICENSE.txt`, `STB_TRUETYPE-H-DUAL-LICENSE.txt` |
| PlutoVG bundled stb image writer | `PLUTOVG-STB-IMAGE-WRITE-DUAL-LICENSE.txt` |

## Notes

The top-level `LICENSE.txt` describes the project licence and points to this directory.
Third-party source files may still contain their original licence headers or licence files; those should not be removed casually because many licences require preservation of notices.
